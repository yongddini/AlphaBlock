# WAN-151 §2 — 분리 존에서의 부품 분해 (WAN-114/145 사다리 재검)

창을 **2023-07-14 ~ 2026-07-15** 로 못 박은 6심볼 × 2TF(15m·1h) × IS/OOS 격자, 렌즈는 **`baseline` 단독**(WAN-128).

재현: `uv run python -m backtest.wan151_entry_ablation` (요약만: `--from-csv`). 원자료: `backtest/reports/wan151_entry_ablation.csv`.

## 이 리포트가 검정한 엔진

채택 기본값 — `entry_mode=zone_limit, rsi_mode=realtime, rsi_gate_mode=unconditional, retap_mode=every_tap, zone_limit_offset_bps=2.0, take_profit_r=1.5, band_bar=intrabar_live, combine_obs=False`. **존 정의를 고정하지 않는다**(WAN-114/145는 `LEGACY_OB_PARAMS`로 병합에 고정돼 있다) — 그게 이 표의 존재 이유다.

🚨 **`L2`는 채택 기본값이 아니다 — 이 표에서 채택 기본값은 `L2u`다.** 사다리 정의는 WAN-145에서 그대로 import했고(라벨과 설정이 갈라지지 않게), 움직인 축은 **존 하나**다.

## 검산

`L2u`(채택 기본값 단) 24행 ≡ §1 널의 `long_only` 실제 팔 — 수익·거래 수 차이 0. 두 새 모듈이 서로를 검산한다.
