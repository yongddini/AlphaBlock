# WAN-120: 엄격 인과 밴드 — 잔여 1분 룩어헤드가 WAN-119의 판정을 흔드나

재현: `uv run python -m backtest.wan120_strict_causal_band` (요약만 재생성: `--from-csv`)

창을 **2023-07-14 ~ 2026-07-15** 로 못 박은 6심볼 격자다(WAN-111/114/115/119와 같은 창). 토대(존 지정가 + 오프셋 2bp · 공식 렌즈 `baseline` · 비용 현재값 · 롱 온리)를 고정 입력으로 받고 **밴드의 20번째 표본만** 바꾼다.

⚠️ **기본값은 바꾸지 않았다**(`band_bar="tap"` · `ALPHABLOCK_LIVE_TRADING=false`). 측정 이슈이며 진입가 정본 전환은 **사용자 결정**이다 — `docs/decisions/wan120.md`.

## 판정 — 공식 렌즈(`baseline`) OOS

- **15m oos**: 채택 기본값 절대 수익률이 `intrabar_live` **+4.41%** → `intrabar_causal` **+2.49%**(3/6심볼, **-1.93%p**) · 볼린저 증분 **+16.70%p** → **+14.77%p** → **유지**
- **1h oos**: 채택 기본값 절대 수익률이 `intrabar_live` **-0.41%** → `intrabar_causal` **-1.22%**(2/6심볼, **-0.81%p**) · 볼린저 증분 **-2.80%p** → **-3.62%p** → **유지**

판정 기준은 **부호**다(크기가 아니라) — 이슈가 물은 것은 「잔여 1분이 판정을 흔드나」이지 「어느 밴드가 더 버나」가 아니다. 수익으로 밴드를 고르면 최적화이고, 진입가 정본은 **정확성** 문제다.

### 렌즈 안정성 (세 렌즈 전부에서 부호가 유지되나)

- **15m oos**: `baseline` +4.41% → +2.49% · `pen_5bp` +1.30% → -0.37% ⚠️부호반전 · `pen_5bp_drop_50` -0.29% → -3.10%
- **1h oos**: `baseline` -0.41% → -1.22% · `pen_5bp` -2.86% → -3.76% · `pen_5bp_drop_50` -3.44% → -3.60%

### 심볼별 잔여분 (방향이 갈리나, 한쪽으로 몰리나)

- **15m oos**: 인과 쪽이 나은 심볼 **2/6** (평균 -1.93%p) — BNB -2.16%p · BTC +1.52%p · ETH -6.90%p · SOL -2.86%p · TRX -1.38%p · XRP +0.23%p

### 손상은 선별인가 가격인가

- **15m oos** `L2c` vs `L2i`: 수익 **-1.93%p**인데 거래 수는 **+1.7%** (체결률 +0.39%p) — **가격** 쪽 손상
